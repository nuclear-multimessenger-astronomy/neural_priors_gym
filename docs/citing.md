# Citing

If you use `package_name` in your work, please cite the following paper(s):

```bibtex
@article{your_citation_key,
  title   = {Your Paper Title},
  author  = {Last, First},
  journal = {Journal Name},
  year    = {2025},
  doi     = {YOUR_DOI},
}
```

See [`CITATION.cff`](../CITATION.cff) at the repository root for a machine-readable citation file.
